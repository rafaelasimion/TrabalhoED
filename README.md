# 📚 Sistema de Cadastro de Alunos

Projeto desenvolvido como parte da disciplina de **Estrutura de Dados**, com o objetivo de aplicar conceitos como **arrays**, **ordenação**, **filtros** e **interação com o DOM**.

## 🎯 Objetivo

Simular um sistema de cadastro de alunos, permitindo registrar até 10 alunos e gerar relatórios com base em critérios diversos.

## ⚙️ Funcionalidades

- Cadastro de alunos com validações (nome, RA, idade, média, sexo)
- Ordenação crescente por **nome** (Selection Sort)
- Ordenação decrescente por **RA** (Bubble Sort)
- Filtro de **aprovados** com ordenação por nome
- Interface simples com botões para cada operação
- Exibição dos dados em **tabelas estilizadas**

## 💡 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript puro

## 🧠 Conceitos de Estrutura de Dados aplicados

- Vetores (arrays de objetos)
- Algoritmos de ordenação:
  - **Selection Sort** (por nome)
  - **Bubble Sort** (por RA)
- Filtragem de dados (`.filter()`)
- Manipulação do DOM e eventos
